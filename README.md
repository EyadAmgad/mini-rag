# mini-rag
RAG implementation Using AI-Agents (Agentic RAG).

## Overview
`mini-rag` is a lightweight Retrieval-Augmented Generation (RAG) project that leverages AI agents to improve contextual reasoning and dynamic knowledge retrieval.  
It demonstrates how agentic workflows can be combined with LLMs to provide accurate and explainable responses.

## Installation
Clone the repo and set up the environment:

```bash
$ git clone https://github.com/your-username/mini-rag.git
```
Then go to repo dirctory 
```bash 
$ cd mini-rag
```
choose your environment 
```bash
$ conda create -n mini-rag python=3.10 -y
$ conda activate mini-rag
```
Install the required  dependencies 
``` bash
$ pip install -r requirements.txt
```
